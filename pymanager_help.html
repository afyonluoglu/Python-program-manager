Python Program Yöneticisi'ne Hoş Geldiniz

Python projelerinizi organize edin, çalıştırın ve yönetin

ℹ️
Hızlı Başlangıç: Program açıldığında son kullanılan klasör otomatik olarak yüklenir. Yeni bir klasör seçmek için Ctrl+O tuşlarını kullanın veya Dosya → Klasör Seç... menüsünü tıklayın.

✨ Temel Özellikler

  • Python dosyalarını (.py) kolayca çalıştırma ve düzenleme
  • ZIP arşivlerini görüntüleme, çıkartma ve sıkıştırma
  • JSON ve Markdown dosyaları için özel görüntüleyiciler
  • Gelişmiş Python kod analizi (sınıflar, fonksiyonlar, bağımlılıklar)
  • Tema desteği ve özelleştirilebilir arayüz
  • Favoriler listesi ve çalıştırma geçmişi
  • Hariç tutma listesi ile gereksiz dosyaları filtreleme

Dosya İşlemleri

Dosyalarla yapılabilecek işlemler ve context menüsü

👆 Çift Tıklama İşlemleri

  • .py dosyaları: Python yorumlayıcısı ile çalıştırılır
  • .exe dosyaları: Onay alındıktan sonra çalıştırılır
  • .zip dosyaları: ZIP içerik görüntüleyicisi açılır
  • .json dosyaları: JSON görüntüleyicisi açılır
  • .md dosyaları: Markdown görüntüleyicisi açılır
  • .mp3 dosyaları: Dahili ses oynatıcısı ile çalınır
  • Klasörler: Klasörün içine girilir

🖱️ Sağ Tık Menüsü (Context Menu)

  • Çalıştır: Dosyayı çalıştırır
  • Düzenle: Python dosyasını editörde açar
  • Açıklama Düzenle: Dosyaya açıklama ekler (sadece .py)
  • Python Proje Analizi: Detaylı kod analizi yapar
  • Favorilere Ekle/Kaldır: Favori listesine ekler veya çıkarır
  • EXE'ye Çevir: Python dosyasını çalıştırılabilir yapar
  • ZIP'e Sıkıştır: Dosya veya klasörü sıkıştırır
  • Dosyaya Git: Sistem dosya gezgininde gösterir
  • Yeniden Adlandır: Dosya adını değiştirir (F2)
  • Sil: Dosyayı siler (Delete)
⚠️
Dikkat: Silme işlemi geri alınamaz! Dosya kalıcı olarak silinir, Geri Dönüşüm Kutusu'na gönderilmez.

ZIP Dosya Yönetimi

Arşiv dosyalarını görüntüleme, oluşturma ve çıkartma

📦
ZIP İçerik Görüntüleyici GELİŞTİRİLDİ

ZIP dosyalarına çift tıklayarak içeriği ağaç yapısında görüntüleyin. İçerideki dosyalara erişmek için sağ tık menüsünü kullanın.

👁️ ZIP İçeriği Görüntüleme

  • ZIP dosyasına çift tıklayarak içerik görüntüleyiciyi açın
  • İçerik ağaç yapısında, klasörler ve dosyalar ayrı ayrı gösterilir
  • Her dosyanın boyutu ve sıkıştırılmış boyutu görüntülenir

🖱️ ZIP İçi Dosya İşlemleri

  • Çalıştır: Dosyayı geçici olarak çıkartıp çalıştırır
  • Çıkart...: Dosyayı istediğiniz konuma çıkartır
  • Python Dosyasını Aç: READONLY Python dosyalarını editörde güvenli modda görüntüler

📁 Klasör/Dosya Sıkıştırma

  • Dosya veya klasöre sağ tıklayıp "ZIP'e Sıkıştır" seçeneğini kullanın
  • ZIP dosyası aynı dizinde, aynı isimle oluşturulur
  • Hariç tutulan dosyalar otomatik olarak ZIP'e eklenmez
💡
İpucu: ZIP sıkıştırma işleminde __pycache__, .git gibi gereksiz klasörler otomatik olarak hariç tutulur. Bu ayarı Ayarlar → Genel Ayarlar menüsünden özelleştirebilirsiniz.

Python Editor

Entegre kod editörü ile Python dosyalarını düzenleme

🖥️ Editör Özellikleri

  • Syntax Highlighting: Python kodu renkli olarak gösterilir
  • Satır Numaraları: Her satırın numarası görüntülenir
  • Kaydet: Değişiklikleri kaydedin (Ctrl+S)
  • Çalıştır: Dosyayı doğrudan editörden çalıştırın
🔒
Readonly (Salt Okunur) Mod READONLY

ZIP içindeki Python dosyalarını güvenli şekilde görüntüleyin. Bu modda:

  • Dosya içeriği değiştirilemez
  • Kaydet butonu gizlenir
  • Syntax highlighting aktif kalır
  • İşlem geçmişe kaydedilir

Python Proje Analizi

Kod yapısı, bağımlılıklar ve istatistikleri analiz etme

ℹ️
Python Proje Analizi özelliğine erişmek için bir .py dosyasına sağ tıklayın ve "Python Proje Analizi" seçeneğini seçin.

📊 Analiz Özellikleri

  • Proje Keşfi: Ana dosyadan başlayarak tüm bağımlı dosyaları otomatik bulur
  • Kod İstatistikleri: Toplam satır, kod satırı, yorum satırı, boş satır sayıları
  • Yapısal Analiz: Sınıflar, metotlar ve fonksiyonların listesi
  • Import Analizi: Standart kütüphane, üçüncü taraf ve kullanıcı tanımlı importlar

📑 Analiz Sekmeleri

  • Proje Özeti: Genel bilgiler ve özet istatistikler
  • Dosyalar: Projedeki tüm dosyaların listesi ve detayları
  • İmportlar: Kategorize edilmiş import listesi
  • İstatistikler: Detaylı kod metrikleri
  • Yapısal Analiz: Sınıf ve fonksiyon hiyerarşisi
  • Hatalar: Analiz sırasında bulunan sorunlar (varsa)

Hariç Tutma Ayarları (Exclusion List)

Belirli dosya ve klasörleri işlemlerden hariç tutma

YENİ
✨
Yeni Özellik! Hariç tutma listesi ile gereksiz dosyaları filtreleyerek daha temiz ve hızlı çalışma deneyimi yaşayın.

🔧 Ayarlara Erişim

Ayarlar → Genel Ayarlar... menüsünden hariç tutma listesine erişebilirsiniz.

📝 Pattern Tanımlama

Virgülle ayrılarak birden fazla pattern tanımlayabilirsiniz:

Pattern Açıklama Örnek Eşleşmeler
__pycache__ Python cache klasörleri src/__pycache__/, lib/__pycache__/
*.pyc Derlenmiş Python dosyaları module.pyc, utils.pyc
.git Git versiyon kontrol klasörü proje/.git/
build Derleme çıktı klasörü proje/build/
dist Dağıtım klasörü proje/dist/
dist\*.* dist klasöründeki tüm dosyalar dist/app.exe, dist/readme.txt
node_modules Node.js bağımlılık klasörü proje/node_modules/
*env\*.* Sanal ortam klasörleri venv/, .env/, myenv/
*.egg-info Python paket bilgi klasörleri mypackage.egg-info/

⚡ Etkilenen İşlemler

Bir klasörü ZIP dosyasına sıkıştırırken, hariç tutma listesine uyan dosya ve klasörler otomatik olarak atlanır.

  • Önce klasör taranır ve hariç tutulan öğeler belirlenir
  • Hariç tutulan klasörlerdeki toplam dosya sayısı hesaplanır
  • Kullanıcıya kaç dosyanın hariç tutulacağı bildirilir
  • Onay alındıktan sonra sıkıştırma işlemi başlar
💡
Örneğin, bir Python projesini sıkıştırırken __pycache__ klasörleri ve *.pyc dosyaları otomatik olarak ZIP'e eklenmez.

Klasör özelliklerini görüntülerken (sağ tık → Klasör Özellikleri), hariç tutulan dosyalar hesaplamalara dahil edilmez:

  • Dosya Sayısı: Yalnızca dahil edilen dosyalar sayılır
  • Toplam Boyut: Hariç tutulan dosyaların boyutu eklenmez
  • Kod Satır Sayısı: Hariç tutulan .py dosyaları sayılmaz

Dosya araması ve kelime araması sonuçlarında, hariç tutma listesine uyan dosyalar görsel olarak işaretlenir:

  • Eşleşen dosyalar açık kırmızı arka plan ile gösterilir
  • Bu dosyalar hâlâ sonuçlarda görünür, ancak normalde işlenmediği görsel olarak belirtilir
  • Çift tıklayarak bu dosyalara da erişilebilir

🔬 Teknik Detaylar

Hariç tutma sistemi ExclusionManager sınıfı tarafından yönetilir ve şu özellikleri sunar:

  • Pattern Ayrıştırma: Klasör ve dosya pattern'leri otomatik olarak ayrıştırılır
  • Fnmatch Desteği: * ve ? joker karakterleri desteklenir
  • Büyük/Küçük Harf: Karşılaştırmalar büyük/küçük harf duyarsızdır
  • Yol Normalizasyonu: Windows ve Unix yol ayırıcıları otomatik dönüştürülür
# Örnek Kullanım
manager = ExclusionManager("__pycache__, *.pyc, dist\\*.*")

# Klasör kontrolü
manager.should_exclude_dir("__pycache__")  # True
manager.should_exclude_dir("src")          # False

# Dosya kontrolü
manager.is_file_excluded("test.pyc", full_path, root)  # True
⚠️
Not: Hariç tutma ayarları değiştirildiğinde, değişiklikler hemen aktif olur. Programı yeniden başlatmanıza gerek yoktur.

İpuçları ve Püf Noktaları

Programı daha verimli kullanmak için öneriler

🚀 Performans İpuçları

  • Büyük klasörler için lazy loading kullanılır, ilk açılış biraz zaman alabilir
  • Hariç tutma listesini doğru yapılandırarak gereksiz dosyaları filtreleyebilirsiniz
  • node_modules, __pycache__ gibi büyük klasörleri hariç tutun

💾 Otomatik Kayıt

  • Pencere boyutları ve konumları otomatik kaydedilir
  • Sütun genişlikleri klasör bazında hatırlanır
  • Son açılan klasör program başlangıcında yüklenir
  • Tema ayarları kalıcı olarak saklanır

🛡️ Güvenlik

  • .exe dosyaları çalıştırılmadan önce onay istenir
  • ZIP içindeki Python dosyaları salt okunur modda açılır
  • Silme işlemleri geri alınamaz, dikkatli olun

🔧 Sorun Giderme

  • Dosya listesi yüklenmiyor: F5 ile yenileyin
  • Pencere konumu bozuk: Ayarlar → Pencere Ayarları'ndan sıfırlayın
  • Python çalışmıyor: Python'un PATH'e eklendiğinden emin olun
  • Karakter sorunu: UTF-8 encoding kullanıldığından emin olun
