Klasör Seçme Gezinme Dosya İşlemleri ZIP Yönetimi Python Editor Favoriler Arama Geçmiş Temalar Pencere Ayarları Kısayollar

1. Klasör Seçme Ctrl+O

  • Program açıldığında son kullanılan klasör otomatik yüklenir.
  • Yeni klasör seçmek için Dosya > Klasör Seç... veya Ctrl+O kullanın.
  • Seçilen klasör ve alt klasörler sol panelde ağaç yapısı olarak gösterilir.

2. Gezinti

  • Sol panelde klasör ağacını genişletmek için klasör adının yanındaki oku tıklayın.
  • Klasör seçildiğinde sağ panelde yalnızca .pyPY, .exeEXE, .zipZIP ve .mp3MP3 dosyalar listelenir.
  • Üst klasöre dönmek için listede .. satırını çift tıklayın.

3. Dosya İşlemleri

  • Dosyaya çift tıklama ile şu işlemler yapılır:
    • .py: Python yorumlayıcısıyla çalıştırılır.
    • .exe: Doğrudan çalıştırılır.
    • .zip: İçerik görüntüleme penceresi açılır.
    • .mp3: Dahili oynatıcı ile çalınır.
  • Hata oluşursa, hata mesajı ayrı pencerede gösterilir.
  • Sağ tık menüsünde ek seçenekler:
    • Açıklama Düzenle (sadece .py)
    • Çalıştır / Çevir > EXE
    • Favorilere Ekle / Kaldır
    • ZIP Adı Değiştir
    • Dosyaya Git (sistem dosya gezgini)
    • Sil (geri alınamaz!)

4. ZIP Dosya Yönetimi

  • ZIP İçeriği Görüntüleme: ZIP dosyalarına çift tıklayarak içeriği ağaç yapısında görüntüleyin.
  • Dosya Çalıştırma: ZIP içindeki dosyalara çift tıklayarak geçici olarak çıkartıp çalıştırın.
  • Sağ Tık Menüsü:
    • Çalıştır: Dosyayı geçici olarak çıkartıp çalıştır
    • Çıkart...: Dosyayı istediğiniz konuma çıkart
    • Python Dosyasını AçREADONLY: Python dosyalarını editörde güvenli modda görüntüle
  • Platform Desteği: Windows, macOS ve Linux'ta ZIP işlemleri desteklenir.
  • Geçici Dosya Yönetimi: Çıkartılan dosyalar otomatik olarak temizlenir.

5. Python Editor İyileştirmeleri

  • Readonly (Salt Okunur) ModREADONLY:
    • ZIP içindeki Python dosyalarını güvenli şekilde görüntüleyin
    • Syntax highlighting korunur
    • Dosya düzenleme engellenir
    • Kaydet butonu otomatik gizlenir
  • Gelişmiş Pencere Yönetimi:
    • Modal pencere kontrolü
    • Otomatik focus yönetimi
    • Pencereyi en üste getirme
  • Geçmiş Entegrasyonu: Readonly editör kullanımı geçmişe kaydedilir

6. Favoriler Paneli Ctrl+B

  • Görünüm > Favoriler Panelini Göster/Gizle veya Ctrl+B.
  • Favorilere eklemek için dosyaya sağ tık > Favorilere Ekle.
  • Panelde çift tıklama ile çalıştırma, sağ tık ile düzenleme veya kaldırma.
  • Panel konumu ve görünürlüğü kaydedilir.

7. Arama

  • Dosya Ara: Ctrl+F
  • Kelime Ara: Ctrl+Shift+F
  • Arama sonuçlarında çift tıklama ile çalıştırma veya dosya açma.

8. Geçmiş İşlemler Ctrl+H

  • Yapılan işlemler (çalıştırma, sıkıştırma, arama, yeniden adlandırma, MP3 oynatma) kaydedilir.
  • Filtre: İşlem türüne göre renk kodlu checkbox'larla süzme.
  • Tarih aralığı (Bugün, Dün, Son Hafta, Son Ay, Hepsi) filtresi.
  • İstatistikler: İşlem sayıları ve en çok çalıştırılan program listesi.
  • Yeni Kayıt TürleriYENİ:
    • ZIP dosya çıkartma işlemleri
    • Python Editor (Readonly) kullanımı
    • ZIP içerik görüntüleme

9. Temalar Ctrl+T

  • Tema Yönetimi penceresinden renk paletleri oluşturun veya düzenleyin.
  • Kaydedilen tema program açılışında otomatik uygulanır.
  • Varsayılana Dön seçeneği ile orijinal tema yüklenir.

10. Pencere Ayarları Ctrl+P

  • Tüm pencerelerin (ana, geçmiş, tema, arama, ZIP içeriği, yardımcı) boyut ve konumu kaydedilir.
  • Ayarlar > Pencere Ayarları'dan her pencereyi varsayılan konuma sıfırlayın.
  • Yeni Pencere TürleriYENİ:
    • ZIP İçeriği pencereleri
    • Python Editor (Readonly) pencereleri
    • Modal pencere yönetimi

11. Klavye Kısayolları

  • Ctrl+O: Klasör Seç
  • Ctrl+D: Dosya Ara
  • Ctrl+F: Kelime Ara
  • Ctrl+B: Favoriler Paneli
  • Ctrl+H: Geçmiş İşlemler
  • Ctrl+T: Tema İşlemleri
  • Ctrl+P: Pencere Ayarları
  • F1: Yardım
  • Delete: Seçili dosyayı sil
  • Esc: Açık pencereleri kapat

📄 Dosya Türü Desteği

🐍 Python Dosyaları (.py)

  • Çift Tıklama: Python dosyasını çalıştırır
  • Sağ Tık Menüsü: Çalıştır, Düzenle, Bağımlılık Analizi seçenekleri
  • Renk Kodu: Kırmızı renkte görüntülenir
  • Açıklama Ekleme: Dosya açıklaması ekleyebilirsiniz

📄 JSON Dosyaları (.json)

  • Çift Tıklama: JSON görüntüleyici açılır
  • Özel Görüntüleyici: Formatlanmış JSON içeriği
  • Renk Kodu: Turuncu renkte, açık turuncu arka plan
  • Özellikler: Kopyalama, varsayılan uygulama ile açma

📖 Markdown Dosyaları (.md, .markdown)

  • Çift Tıklama: Markdown görüntüleyici açılır
  • Formatlanmış Görünüm: Başlıklar, kod blokları, listeler formatlanır
  • Renk Kodu: Mavi renkte, GitHub benzeri arka plan
  • Ham/Formatlanmış Geçiş: İki görünüm modu arasında geçiş

📁 Klasörler

  • Renk Kodu: Kahverengi arka plan, altın sarısı metin
  • Gezinme: Çift tıklayarak klasöre giriş
  • Üst Klasör: ".." ile bir üst klasöre çıkış

⚙️ Çalışabilir Dosyalar (.exe)

  • Renk Kodu: Yeşil renkte görüntülenir
  • Çalıştırma: Güvenli çalıştırma onayı ile
